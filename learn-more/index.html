How YOU can create a webpage

(even for free)

First, the code

Start by obtaining the code for the webpage. The code will include an index.html file. Other files may be included though. If you are familiar with HTML and CSS, you can code it yourself. If not, use a website creator that allows you to obtain the code. Many exist, you could use Chat GPT, even. I use mobrise.

Then, the method

Now you have your website file, look at all the ways to host a webpage. You could host it on your own hardware without port forwarding or paying for a web address by using Noip as a static and free url, Nginx to actually host the site and Ngrok as the meathod to host without port forwarding. Or, you can use a service online, this can be free or paid. Paid plans include hostinger or wix paid plan. Free plans include github, but requires technical knowledge or google sites which does not allow importing of the raw files.

Take action

Then, publish the website. On github pages, create a repo called <yourusername>.github.io, but replace "<yourusername>" with whatever your username may be. Then upload all your website files to it.
Then, if the repo is public, your website will be hosted at the name of your github repo. In my case, troy-davies.github.io.
